===РЕЛИЗ===
***Исправления***
- Неработоспособность базы данных одежды на ReactOS и Windows XP, вызванная принудительной сменой родительской директории со стороны файлового менеджера (критический баг)
- При повреждении второй строки конфигурационного файла программы (setting.yml), проверка обновлений автоматически отключается
- Некоторые иконки, чрезмерно высокого разрешения, вызывали артефакты на Windows XP и ReactOS
- Дата и время создания (редактирования) элементов БД записывалась некорректно
- При отмене редактирования изображения БД одежды, журнал сохраняет редактирование как успешное
- При переходе на другую страницу, состояние кнопок "изменить изображение" и "просмотр изображения" не обновляется
- Некорректный порядок элементов tab в лобби
- При использовании поиска в редакторе элементов БД, форма поиска в главной форме БД не обновляется
- При попытке открыть рез. копию для общего доступа, исключение при отказе доступа
- Иконка кнопок обновления страницы и поиска по базе данных имеет чрезмерно высокое разрешение, в результате на Windows XP и ReactOS отображается некорректно
- Некорректный порядок элементов в режиме отладки

***Изменения***
+ Двойной клик по элементу таблицы БД вызывает редактор и выделяет элемент в списке
+ Редактор БД инвентаря и личного состава теперь можно развернуть во весь экран
+ Переход к .NET Framework 4.0
+ Работоспособность на Windows XP и ReactOS
+ Табличная интерпритация баз данных
+ Прирост производительности
