***ХОТ-ФИКС***
+ Исправлен баг с ошибочным количеством элементов в редакторе объектов БД

***Исправления***
- Возникновение исключения при попытке открыть резервную копию для общего доступа
- Глюки, связанные с чрезмерно высоким разрешением изображений некоторых кнопок
- Зависание програмы при недостаточной скорости интернет-соединения для проверки обновлений во время запуска
- Некорректный порядок элементов в режиме отладки
- Неработоспособность системы исправления целостности учетных записей

***Изменения***
+ Теперь автоматическую проверку обновлений можно отключить в меню "Настройки"
+ Новая система хранения объектов БД (совместима с предыдущими версиями) позволяющая снять ограничения на ввод аргументов (размер строки, тип символа)
+ Новая реализация редактора объектов БД
+ Введена система контроля проверки обновлений для отключения таковой, при низкой скорости интернет-соединения
+ Размер динамических рабочих окон теперь уникален для каждой учетной записи
