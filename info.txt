KeepInventory предназначена для ведения учёта инвентаря и личного состава. Объектная база данных позволяет добавлять объекты со следующими аргументами: наименование, количество, ид/штрих-код, примечания; Личная база данных: ФИО, и три пункта "заметки". Каждый из аргументов может быть введён любым способом (буквы, цифры, символы).
KeepInventory is designed to keep inventory of inventory and personnel. The object database allows you to add objects with the following arguments: name, quantity, id / barcode, notes; Personal database: Name, and three items of the "note". Each of the arguments can be entered in any way (letters, numbers, symbols).
